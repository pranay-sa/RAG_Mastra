RAG System with TOON serialization - Branch Mastra 

An implementation of a Retrieval-Augmented Generation (RAG) system built with Mastra - Typescript Framework 

Architecture :
Divided into 3 parts : 
1) Agent - Mistral LLm to generate responses from the retreivals
2) Tools - contains documentParser (PDF+OCR) , chunking , embedding , vectorStores and Retreival+Reranking
3) WorkFlow - combines all the abvoe tools in sequential order
